# Life Hack or Myth App

## Overview
This Android application was developed using Kotlin in Android Studio. The purpose of the app is to help users distinguish between real-life hacks and common myths through an interactive flashcard quiz.

## Features
- Welcome screen with start button
- Quiz screen with multiple questions
- Immediate feedback for answers
- Score calculation and result display
- Review screen with explanations

## Technologies Used
- Kotlin
- Android Studio
- ConstraintLayout
- Material Design Components

## Logging
Logging was implemented using Log.d() to track:
- App startup
- User answers
- Score updates
- Navigation between screens

## Testing
The application was tested manually by:
- Navigating through all screens
- Verifying correct score calculation
- Ensuring smooth transitions

## How to Run
1. Clone the repository
2. Open in Android Studio
3. Run on emulator or Android device
